# BI-Clinica
Dashboard ejecutivo de una sola página con 3 vistas (bookmarks) para una clínica dermatológica ficticia. El proyecto cubre el ciclo completo: ingesta y limpieza de 24,115 registros transaccionales en Power Query, modelo estrella con 12 tablas y 13 relaciones, 29 medidas DAX en 7 carpetas temáticas, segmentación RFM de 1,195 clientes con Python y forecast mensual con Facebook Prophet. Los resultados muestran una venta neta de S/ 25.6M en 24 meses, crecimiento +7.4% interanual y una proyección de S/ 3.6M para el primer semestre 2026. El objetivo fue demostrar un flujo de analítica avanzada end-to-end — desde datos crudos hasta insights accionables — usando herramientas del stack moderno de BI.
